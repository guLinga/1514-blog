题目：
```
const arr = []
arr[(2 ** 32) - 2] = 1
arr[(2 ** 32) - 1] = 2
console.log(arr.at(-1))
```

答案：
```
1
```

解析：
1. 数组的储存限制是2的32次幂减2
2. 2 ** 32为2的32次幂
3. arr.at(val)，如果val是正数，则查找arr数组中val索引的值，如果为负数则从末尾往前查找。val不输入默认为0，如果val超出arr的长度，则返回undefined，正着查找的第一个数val为0，而从末尾查找的第一个数val为-1
